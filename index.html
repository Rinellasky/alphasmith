How to use these alphas

Blender (sculpt)

In Sculpt Mode open Brush Settings → Texture → New, load the PNG, and set Mapping to Area Plane (or Stencil). Use the Draw or Clay brush with stroke method Anchored or Drag Dot to stamp the shape. For displacement, use the 16-bit PNG in a Displace modifier with a non-color image.

Unreal Engine (landscape)

Import the PNG, set Compression to Grayscale and turn sRGB off. In Landscape Sculpt mode, set Brush Type to Alpha (or Pattern) and assign the texture. The 16-bit export keeps slopes smooth on large landscapes.

Photoshop / Krita

Open the PNG and choose Edit → Define Brush Preset. Dark areas paint, white stays transparent — hit the Invert toggle before downloading if you want the opposite. In Krita, use Predefined brush tips → Import.
